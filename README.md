# koikeasuka-claude-plugins

koikeasuka 個人用の Claude Code プラグイン置き場。

## インストール

```
/plugin marketplace add koikeasuka/claude-plugins
```

その後 `/plugin` でプラグインマネージャーから個別にインストール。

## 収録プラグイン

| 名前 | 説明 |
|---|---|
| spike-support | ジュニアエンジニアのスパイク打ち（実装前の小さな検証作業）をサポート |
| quiz | git diff のコード変更を使った復習のためのクイズ |
